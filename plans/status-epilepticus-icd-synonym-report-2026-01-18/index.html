    Code Selection Guide¶

    For new presentation: - Use G40.901 (not intractable, with SE) for first-time SE - Use G41.0/G41.1/G41.2 if seizure type is clearly identified

    For known epilepsy patients: - Use the specific epilepsy code with SE modifier (5th character "1") - Use "intractable" codes (G40.x11) if patient has drug-resistant epilepsy

    For NORSE/FIRES: - Primary: G41.8 (Other status epilepticus) - Add: G04.81 if autoimmune encephalitis confirmed

## Plan Structure

Each plan follows a standardized 8-section format:

**Section A: Action Items**

1. Laboratory Workup (Core, Extended, Specialized)
2. Diagnostic Imaging & Studies
3. Treatment (Acute, Symptomatic, Maintenance, Disease-Modifying)
4. Other Recommendations (Referrals, Patient Instructions, Lifestyle)

**Section B: Reference**

5. Differential Diagnosis
6. Monitoring Parameters
7. Disposition Criteria
8. Evidence & References

### Priority Key

| Priority | Meaning |
|----------|---------|
| STAT | Immediate |
| URGENT | Within hours |
| ROUTINE | Standard timing |
| EXT | Extended/atypical cases |
| - | Not applicable |

### Setting Columns

- **ED** - Emergency Department
- **HOSP** - Hospital/Inpatient
- **OPD** - Outpatient/Clinic
- **ICU** - Intensive Care Unit
